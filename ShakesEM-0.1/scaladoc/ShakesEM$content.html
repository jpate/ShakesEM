Class Summary
class ShakesConvergenceManager (toParse : scala.List[java.lang.String], g : ShakesPCNF, numParsers : Int, tolerance : Double, gramFilePrefix : java.lang.String) extends AnyRef
Run EM until convergence (change in log probability of a corpus after an iteration is less than some tolerance)
class ShakesIterationManager (toParse : scala.List[java.lang.String], g : ShakesPCNF, numParsers : Int, numIterations : Int, gramFilePrefix : java.lang.String) extends AnyRef
Run EM a specific number of iterations.
class ShakesPCNF extends AnyRef
ShakesPCNF defines a Probabilistic Chomsky Normal Form grammar for use with the ShakesEM library
class ShakesParser (id : Int, g : ShakesPCNF) extends scala.actors.Actor
This is an Actor for distributing the sentences from one corpus over as many cores as you happen to have access to. This should normally be started and managed through a ShakesParserManager.
class ShakesParserManager (toParse : scala.List[java.lang.String], g : ShakesPCNF, numParsers : Int, gramFilePrefix : java.lang.String, stoppingCondition : (Int, Double) => Boolean) extends scala.actors.Actor
Starts up a bunch of parsers and farms out the sentences to them for an iteration, updates the grammar, and repeats until the stopping condition is reached.
