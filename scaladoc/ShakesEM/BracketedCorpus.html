Reads in strings of terminals from stringsPath and the brackets from bracketsPath. stringsPath should point to a file with one sentence per line. bracketsPath should point to a file that specifies the partial bracketing with each span specified in the following format:

sentence-id span-start-index span-end-index

Where sentence-id is the zero-based index of the sentence in the stringsPath. Basically, I think it is just easier to get a perl script to take a Penn-Treebank style corpus and produce these files than to do it in Scala.
