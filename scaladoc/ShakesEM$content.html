Class Summary
class BracketedCorpus extends ShakesTrainCorpus with AnyRef
Reads, stores, and makes useful a partially bracketed corpus.
abstract class EvaluationActor (initGram : ShakesPCNF, ws : Int) extends ShakesViterbiParser with scala.actors.Actor
class ShakesBracketedParser (id : Int, grammar : ShakesPCNF, ws : Int) extends ShakesEstimatingParser
This implements the extension of standard EM from Pereira and Schabes (1992) by simply overriding the relevant methods from the ShakesParser class.
abstract class ShakesDistributedParser extends ShakesFullCYKParser with scala.actors.Actor
class ShakesEstimatingParser (id : Int, grammar : ShakesPCNF, ws : Int) extends ShakesDistributedParser
This provides a real parser that can provide a parse chart with partial estimates all filled in.
trait ShakesFullCYKParser extends ShakesParser
This provides chart filling definitions for CYK parsing
class ShakesPCNF extends AnyRef
ShakesPCNF defines a Probabilistic Chomsky Normal Form grammar for use with the ShakesEM library
trait ShakesParser extends AnyRef
This defines what a parser must have, without giving an explicit definition for certain functions we might want to change (such as re-estimation functions). This must be extended by providing at least the actual parsing functions synFill, lexFill, and populateChart. Other functions can of course be provided as well, for example computeOPWithEstimates to provide outside probability estimates.
abstract class ShakesParserManager (initGram : ShakesPCNF, trainingCorpus : ShakesTrainCorpus) extends scala.actors.Actor
Create many estimating parsers and estimate a grammar. Use this by implementing useGrammar (this receives the trained grammar after every iteration), stoppingCondition (iterations continue until this returns true), and parserConstructor (builds an array of parsers)
trait ShakesTrainCorpus extends AnyRef
Training corpus classes. Apply should return whatever counts as one sentence (whether it's a string or a tuple of a string and a bracketing), size should give the number of sentences, and readCorpus should read in the corpus from file(s)
class ShakesViterbiParser (grammar : ShakesPCNF, ws : Int) extends ShakesParser
class StringsOnlyCorpus extends ShakesTrainCorpus with AnyRef
Reads, stores, and makes useful a corpus with only strings
