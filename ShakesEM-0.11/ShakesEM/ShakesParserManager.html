class ShakesParserManager(toParse : scala.List[java.lang.String], g : ShakesPCNF, numParsers : Int, gramFilePrefix : java.lang.String, stoppingCondition : (Int, Double) => Boolean)
extends scala.actors.Actor
Starts up a bunch of parsers and farms out the sentences to them for an iteration, updates the grammar, and repeats until the stopping condition is reached.
Parameters
toParse - The sentences to parse.
g - The (initial) grammar to use while parsing.
numParsers - The number of parsers to use.
numIterations - The number of iterations we would like to run for.
gramFilePrefix - The prefix of the path to the file that grammars are
stoppingCondtion - Are we done yet?
